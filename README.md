# Iris_data_analysis
Data analysis and preprocessing of the Iris dataset using Python, including data cleaning, visualization, and feature exploration.


This project focuses on exploratory data analysis and preprocessing of the Iris dataset using Python. The goal is to understand the structure of the data, handle missing values, and prepare it for machine learning applications.

The workflow includes data loading, inspection, handling missing values, feature type identification, data cleaning, and basic statistical analysis. Numerical and categorical features are processed separately, and preprocessing techniques such as median imputation and data type conversion are applied.

This project demonstrates fundamental data analysis skills required in real-world machine learning pipelines, including data preparation, feature understanding, and dataset validation.
